# 人物简介

梁艺馨，20级工高班成员，主修专业为电子科学与技术，计划要前往卡内基梅隆大学（CMU）攻读电子与计算机工程（ECE）研究生。大二时参加创新创业类比赛，获得节能减排大赛全国一等奖；大三前往杜克大学参加暑期研究项目。国家奖学金获得者。
